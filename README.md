# LUTB - Site E-commerce

## Descrição do Projeto

Este projeto é um site e-commerce desenvolvido para a loja **LUTB**, uma cliente que precisava de uma página virtual para apresentar seus produtos de forma organizada, moderna e acessível.

O site foi criado com foco em uma navegação simples, visual agradável e boa apresentação dos itens disponíveis na loja. A proposta é simular uma loja virtual funcional, permitindo que os clientes conheçam melhor os produtos e tenham uma experiência próxima de um e-commerce real.

---

## Objetivo

O principal objetivo do projeto é desenvolver um site de e-commerce para a loja **LUTB**, oferecendo uma presença digital mais profissional para a marca.

O sistema busca facilitar a visualização dos produtos, organizar as informações da loja e melhorar a experiência do cliente ao acessar o catálogo online.

---

## Funcionalidades

O site conta com as seguintes funcionalidades:

- Página inicial com apresentação da loja;
- Exibição de produtos em formato de cards;
- Layout voltado para loja virtual;
- Organização visual dos produtos;
- Área com informações importantes da loja;
- Navegação simples e intuitiva;
- Design responsivo para diferentes tamanhos de tela;
- Interface visual mais profissional para o cliente final;
- Estrutura preparada para futuras melhorias.

---

## Tecnologias Utilizadas

As tecnologias utilizadas no desenvolvimento do projeto foram:

- **HTML5**  
  Utilizado para criar a estrutura das páginas do site.

- **CSS3**  
  Utilizado para a estilização, organização visual, cores, espaçamentos e responsividade.

- **JavaScript**  
  Utilizado para adicionar interatividade e melhorar a experiência do usuário.

---

## Como Executar o Projeto

Para executar o projeto localmente, basta abrir o arquivo principal do site no navegador.

Também é possível utilizar o **Visual Studio Code** com a extensão **Live Server**, o que facilita a visualização das alterações em tempo real durante o desenvolvimento.

---

## Público-Alvo

O site foi desenvolvido para os clientes da loja **LUTB**, que desejam visualizar os produtos disponíveis de maneira rápida, simples e organizada.

A proposta é permitir que o usuário encontre informações sobre os produtos sem dificuldade, com uma navegação direta e objetiva.

---

## Identidade Visual

A identidade visual do projeto foi pensada para representar uma loja virtual moderna e bem organizada.

Foram considerados os seguintes pontos:

- Layout limpo;
- Boa legibilidade;
- Destaque para os produtos;
- Organização dos elementos na tela;
- Visual adequado para e-commerce;
- Interface simples para o usuário final.

---

## Responsividade

O site foi desenvolvido considerando a adaptação para diferentes tamanhos de tela, como computadores, notebooks, tablets e celulares.

A responsividade é importante para garantir que o usuário consiga acessar o site corretamente em diferentes dispositivos, mantendo uma boa experiência de navegação.
